# Attributes and methods

## Attribute

Attributes are the variables defined within a class; they represent the state or characteristics of an object created
from that class. They are also commonly referred to as fields, instance variables, or data members

## Methods

A method is a block of code which only runs when it is called, you can pass data, known as parameters, into a method.
Methods are used to perform certain actions, and they are also known as functions.
